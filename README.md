# 🎇 Motivation

React + TypeScript + Redux + FireBase 기술을 공부하기 위해 시작한 프로젝트 입니다.

#  ✨Friends-Store

- 친구들, 직장 동료들의 정보를 적어서 지인들의 정보를 쉽게 관리 할 수 있다.

# 🛠 기술 스택 

- FE : TypeScript, Redux, Styled-Components	

- DataBase :  FireBase

- Image Storage: Cloudinary

# 🌈 구현 기능 (예정)

-  카테고리별 친구 관리

- 친구 등록/ 삭제 기능 

- 친구 이미지 업로드

- 친구 검색 기능
